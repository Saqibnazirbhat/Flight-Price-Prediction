# Flight Price Prediction

Regression model predicting flight ticket prices from features like
airline, source, destination, departure / arrival times, duration, and
number of stops. Built as a machine learning assignment.

## Contents
- `notebook-kg1.ipynb` — full notebook: feature engineering on
  date/time fields, encoding categorical variables, model training and
  evaluation.
- `train.csv`, `test.csv` — provided dataset.

## Run

```bash
pip install pandas numpy scikit-learn matplotlib seaborn jupyter
jupyter notebook notebook-kg1.ipynb
```
